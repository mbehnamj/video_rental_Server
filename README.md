## Description:

A server-side program for a video rental web app.

## To Run:

- npm install
- set/export mb_jwtPrivateKey
- mongod
- npm start

## Features:

- Authentication/Authorization
- Handling and logging errors
- Unit testing
- API servies
- NoSQL database

## Frameworks/Libraries used:

- Node.js
- Express
- MongoDB/Mongoose
- Jest
- Bootstrap
- Lodash
